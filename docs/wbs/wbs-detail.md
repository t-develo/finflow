# FinFlow WBS詳細 - 家計・サブスク管理アプリ

**プロジェクト名:** FinFlow（家計・サブスク管理アプリ）
**技術スタック:** C# (.NET 8) + バニラJS + SQL Server
**体制:** PM 1名 + PL 1名 + SE 3名（SE-A, SE-B, SE-C）
**スプリント期間:** 1週間（5営業日）× 4スプリント = 合計20営業日
**作成日:** 2026年3月8日

---

## 全体スケジュール

| スプリント | 期間 | テーマ | 主な担当 |
|-----------|------|--------|---------|
| Sprint 0 | Week 1（5日） | 準備・設計 | PL単独 |
| Sprint 1 | Week 2（5日） | コア機能実装 | SE-A, SE-B, SE-C 並行 |
| Sprint 2 | Week 3（5日） | 拡張機能実装 | SE-A, SE-B, SE-C 並行 |
| Sprint 3 | Week 4（5日） | 統合・テスト・仕上げ | 全員 |

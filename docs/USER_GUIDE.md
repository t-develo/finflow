## FinFlow とは

FinFlow は家計・サブスクリプション管理アプリです。日々の支出を記録し、月次レポートやグラフで収支を可視化します。CSV ファイルによる一括インポートや、サブスクリプションの支払い期日通知にも対応しています。

---

## アカウントの作成・ログイン

### 新規登録

1. トップページにアクセスします
2. 「新規登録」リンクをクリックします
3. メールアドレスとパスワード（8文字以上）を入力します
4. 「登録する」ボタンをクリックします
5. 登録が完了するとダッシュボードに移動します

### ログイン

1. ログインページにアクセスします
2. 登録済みのメールアドレスとパスワードを入力します
3. 「ログイン」ボタンをクリックします

### ログアウト

画面右上のメニューから「ログアウト」を選択します。

---

## ダッシュボード

ダッシュボードはアプリのホーム画面です。以下の情報を一覧で確認できます。

### 表示内容

- **今月の支出合計**: 今月の支出総額
- **前月比**: 前月と比較した増減率（赤: 増加、緑: 減少）
- **カテゴリ別内訳グラフ**: 今月の支出をカテゴリごとにドーナツグラフで表示
- **最近の支出**: 直近 5 件の支出一覧
- **PDF レポートダウンロード**: 当月の月次レポートを PDF でダウンロード

### PDF レポート

「PDFレポート」ボタンをクリックすると、当月の支出レポートが PDF ファイルとしてダウンロードされます。

## 支出管理

### 支出の追加

1. 左のサイドバーから「支出一覧」をクリックします
2. 画面右上の「+ 支出を追加」ボタンをクリックします
3. 以下の項目を入力します:
   - **金額**: 支出金額（円単位、1 以上の正の数）
   - **カテゴリ**: プルダウンからカテゴリを選択
   - **日付**: 支出日
   - **説明**: 支出の内容（最大 200 文字）
   - **メモ**: 任意のメモ（最大 500 文字）
4. 「保存する」ボタンをクリックします

### 支出の編集

1. 支出一覧画面で対象の支出の ✏️ ボタンをクリックします
2. 内容を編集して「更新する」ボタンをクリックします

### 支出の削除

1. 支出一覧画面で対象の支出の 🗑️ ボタンをクリックします
2. 確認ダイアログが表示されるので「削除する」をクリックします

### 支出の絞り込み・検索

支出一覧画面の上部にあるフィルター欄を使用します:

- **期間**: 年月でフィルタリング（デフォルトは当月）
- **カテゴリ**: カテゴリでフィルタリング

API を直接使用する場合は以下のクエリパラメーターも利用できます:
- `from` / `to`: 日付範囲（YYYY-MM-DD 形式）
- `minAmount` / `maxAmount`: 金額範囲
- `keyword`: 説明文のキーワード検索
- `page` / `pageSize`: ページネーション（デフォルト: 1ページ 50 件）

## カテゴリ管理

カテゴリは支出を分類するためのラベルです。システムカテゴリ（変更不可）と、ユーザーが作成したカテゴリがあります。

### カテゴリの追加

1. 左のサイドバーから「カテゴリ」をクリックします
2. 「+ カテゴリを追加」ボタンをクリックします
3. カテゴリ名と色（カラーピッカー）を設定します
4. 「保存」ボタンをクリックします

### カテゴリの編集・削除

- **編集**: カテゴリ名の横の ✏️ ボタンから編集できます
- **削除**: 🗑️ ボタンから削除できます（支出が紐づいているカテゴリは削除できません）

### システムカテゴリ

デフォルトで以下のシステムカテゴリが用意されています:
- 食費、交通費、日用品、医療費、娯楽・趣味、その他

システムカテゴリは編集・削除できません。

## サブスクリプション管理

毎月・毎年発生するサブスクリプション費用を管理できます。

### サブスクリプションの登録

1. 左のサイドバーから「サブスク管理」をクリックします
2. 「+ 追加」ボタンをクリックします
3. 以下の項目を入力します:
   - **サービス名**: サービスの名称（例: Netflix、Spotify）
   - **金額**: 請求金額（円単位）
   - **請求サイクル**: monthly（月次）/ yearly（年次）/ weekly（週次）
   - **次回請求日**: 次回の請求予定日
   - **カテゴリ**: 紐づけるカテゴリ
   - **メモ**: 任意のメモ
4. 「保存」ボタンをクリックします

### 支払い期日通知

次回請求日の **3 日前** になると、登録済みのメールアドレスに通知メールが送信されます。

### サブスクリプションの有効・無効

使用していないサブスクリプションは「無効」に設定することで一覧に残しつつ通知を停止できます。

## CSV インポート

銀行の明細 CSV ファイルを一括インポートできます。

### 対応フォーマット

| フォーマット | ヘッダー形式 | 対応銀行 |
|------------|------------|--------|
| 汎用フォーマット | `date,description,amount,categoryId` | - |
| MUFG フォーマット | 三菱UFJ銀行の明細形式 | 三菱UFJ銀行 |
| 楽天フォーマット | 楽天銀行の明細形式 | 楽天銀行 |

### インポート手順

1. 左のサイドバーから「CSV取込」をクリックします
2. CSV ファイルをドラッグ＆ドロップするか、「ファイルを選択」で選択します
3. フォーマットを自動検出します（手動で変更も可能）
4. 「インポート開始」ボタンをクリックします
5. インポート結果（成功件数・スキップ件数・エラー内容）が表示されます

### 制限事項

- 最大 10,000 行まで
- ファイルエンコーディング: UTF-8 または Shift_JIS
- エラーのある行はスキップされます（インポート全体は中断されません）

### 汎用フォーマットの仕様

```csv
date,description,amount,categoryId
2026-03-01,スーパーマーケット,3500,1
2026-03-02,電車代,200,2
```

- `date`: YYYY-MM-DD 形式
- `description`: 支出の説明（必須）
- `amount`: 正の数値
- `categoryId`: カテゴリ ID（省略可）

## レポート・PDF ダウンロード

### 月次レポート

左のサイドバーから「レポート」（またはダッシュボードの PDFボタン）にアクセスします。

年月を指定して以下の情報を確認できます:
- 月次合計金額
- 支出件数
- 1 日平均支出
- カテゴリ別内訳（金額・割合）

### PDF ダウンロード

月次レポート画面またはダッシュボードの「PDF レポート」ボタンをクリックすると、レポートが PDF としてダウンロードされます。

ファイル名形式: `finflow-report-YYYY-MM.pdf`

## 自動分類ルール

キーワードに基づいて支出を自動的にカテゴリに分類するルールを設定できます。

### ルールの設定

1. 設定メニューから「自動分類ルール」にアクセスします
2. 「+ ルールを追加」ボタンをクリックします
3. 以下を設定します:
   - **キーワード**: 説明に含まれるキーワード（最大 200 文字）
   - **カテゴリ**: 自動的に割り当てるカテゴリ
   - **優先度**: 複数のルールが一致した場合の優先順位（数値が小さいほど高優先度）
4. 「保存」ボタンをクリックします

CSV インポート時に自動分類ルールが適用されます。

## よくある質問

**Q: パスワードを忘れました**
A: 現在、パスワードリセット機能は実装されていません。管理者にお問い合わせください。

**Q: 支出を削除したカテゴリの支出はどうなりますか？**
A: 支出が紐づいているカテゴリは削除できません。先に支出のカテゴリを変更してください。

**Q: CSV インポートでエラーが出た行のデータはどうなりますか？**
A: エラー行はスキップされ、他の行は正常にインポートされます。エラー内容はインポート結果画面で確認できます。

**Q: PDF レポートが文字化けする場合は？**
A: PDF は日本語対応フォントで生成されています。PDF リーダーが最新版であることを確認してください。

**Q: データは他のユーザーに見えますか？**
A: いいえ。すべてのデータはログインユーザーごとに完全に分離されています。
